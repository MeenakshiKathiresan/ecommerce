About the project Full stack development

This is the final project done for the course full stack development in Fall 2022. It is a simple ecommerce website where you can search for products, add it to cart, checkout and pay for the products. This project has gmail login and paypal payment integrated. The images are stored in AWS S3 bucket. Currently, the project has over 20 products for demonstration. Views are done with HTML combined with bootstrap styles. Apart from this, adobe XD design is used for 1-2 pages.
